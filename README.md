# 🌿 About Me  
디자인 감각을 **웹 화면 위에 피어나게 하고자 하는 신입 프론트엔드 개발자.**  
작은 디테일이 사용자 경험을 바꾼다고 믿습니다.

감성적인 UI와 자연스러운 인터랙션을 좋아하며,  
사용자에게 ‘편안함’을 주는 화면을 만들고 싶습니다.

현재 프론트엔드를 공부하며  
매일 직접 만들고 기록하며 성장하는 중입니다.
